## 包含功能

| 功能 | 说明 |
|---|---|
| 开始菜单 | 标题、操作说明、闪烁开始提示 |
| 玩家控制 | WASD/方向键 8 方向移动，潜艇朝向跟随 |
| 自动鱼雷 | 自动瞄准最近敌人发射，支持多发鱼雷 |

| 8 种强化 | 射速/伤害/多发/护甲/速度/磁力/回血/声波 |
| HUD | 血量条、经验条、存活时间、波次、击杀数、敌人数 |
| 胜利/死亡结算 | 存活 180 秒胜利，HP 归零死亡，展示统计数据 |
| 重新开始 | R 键或点击即可重新开始 |
| 深海视觉 | 渐变背景、探照灯光晕、漂浮微粒、怪物荧光、粒子效果 |
