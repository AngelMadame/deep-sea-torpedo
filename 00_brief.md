# Demo Brief

需求名称：深海求生（Deep Sea Survivor）
游戏类型：类吸血鬼幸存者（Survivors-like）— 俯视角自动射击 + 生存 + Roguelike 升级选择
目标玩家体验：在漆黑深海中操控潜艇，对抗不断涌来的深海异形生物，体验紧张刺激的弹幕生存与成长快感
Demo 时长：单局 3～5 分钟
核心玩法：玩家操控潜艇在 2D 平面移动，潜艇自动发射鱼雷攻击周围敌人；击杀深海怪物掉落经验球，拾取升级后从随机强化选项中选择一项增强能力；怪物波次随时间递增，存活到时间结束即为胜利
目标平台：Web 浏览器（HTML5 Canvas 2D）
参考游戏：Vampire Survivors、Brotato、HoloCure
必须包含：
- 玩家移动（键盘 WASD / 方向键）
- 自动攻击（鱼雷，固定间隔发射向最近敌人）
- 怪物波次系统（随时间递增难度）
- 经验球拾取 + 升级
- 升级时从 3 个随机选项中选择 1 项强化
- HUD 显示血量、经验条、存活时间、击杀数
- 开始界面 + 死亡结算 / 胜利结算
- 至少 3 种不同的深海怪物类型
明确不做：
- 不做存档 / 读档
- 不做角色解锁 / 永久成长
- 不做多关卡
- 不做联机功能
- 不做成长树 / 天赋系统
- 不做 Boss 战（MVP 阶段）
- 不做音效和 BGM（MVP 阶段）
时间限制：无硬性限制
其他限制：
- 纯 JavaScript + Canvas 2D，零外部依赖
- 单文件或极少文件，可直接浏览器打开运行
- 所有视觉元素使用几何图形绘制（不依赖外部素材）
